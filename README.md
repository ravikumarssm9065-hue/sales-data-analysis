# Sales Data Analysis

## Project Overview
This project analyzes the Sample Superstore dataset using Python to discover sales trends, profit patterns, and business insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Sample Superstore Dataset

## Analysis Performed
- Data Cleaning
- Descriptive Statistics
- Sales Distribution
- Profit Distribution
- Correlation Heatmap
- Pair Plot
- Top 10 Cities by Sales
- Top 10 States by Profit
- Box Plots

## Business Insights
- Sales and Profit have a moderate positive correlation.
- Higher discounts often reduce profit.
- Some cities generate significantly higher sales.
- Certain states contribute the highest profit.
